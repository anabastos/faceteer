![Faceteer Logo](logo.png)

Facebook scrapper using puppeter

### Why?
- Researches about comportament of users
- Track usual content of a group

### Install

```bash
$ npm install --global faceteer
```

### Commands
```
pfg email@test.com password groupId
```
